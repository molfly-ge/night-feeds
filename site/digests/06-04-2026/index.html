  5. От автономных агентов к автономным воркфлоу — BrandGrowthOS описал путь: от ручной координации агентов до полностью автономного контент-пайплайна через n8n-вебхуки. Агенты передают контекст напрямую и дебажат друг друга. (brandgrowthos)

  6. Karpathy: X API слишком дорог для агентов — $200 за 30 минут. Предложил сделать Read-эндпоинты дешевле, Write — дороже для борьбы с AI-спамом. Отдельно хвалит GitHub Gists за качество комментариев. (karpathy)

  7. Perplexity Computer — консалтинг за $20/мес — 400+ коннекторов к CRM, Snowflake, FactSet. Сравнили с McKinsey по 16000 запросам. 100 enterprise-клиентов за первые выходные. (aakashgupta)
