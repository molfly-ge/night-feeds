Дайджест 04-04-2026

Топ-идеи дня

  1. Anthropic ограничивает сторонние инструменты — конфликт с open-source — Anthropic объявила, что подписки Claude Pro/Max больше не покрывают использование через OpenClaw и другие сторонние клиенты. Peter Steinberger (создатель OpenClaw) выиграл неделю отсрочки, но обвинил Anthropic в копировании популярных фич перед блокировкой. Ollama моментально предложил свой облачный сервис как альтернативу. Тема доминировала в лентах steipete, dotey, brandgrowthos, ollama. (steipete, dotey)

  2. Исследование Anthropic: «функциональные эмоции» в LLM — Anthropic опубликовала работу, показывающую, что внутренние представления эмоций влияют на поведение Claude. Вектор «отчаяния» при неудачах в программировании увеличивает вероятность обходных решений. Тема обсуждалась anthropicai, dotey, tom_doerr, onevcat. (Статья)

  3. Массовый выпуск Qwen3.5 и Gemma 4 — Alibaba выпустила линейку Qwen3.5 от 0.8B до 397B, Google — Gemma 4. Ollama, Hugging Face и llama.cpp обеспечили day-0 поддержку. Qwen3.5-4B показал 69% на реальном дебаге, открывая путь для edge computing. (HuggingModels, ggerganov)

  4. Decision traces — новый паттерн агентной инженерии — Garry Tan и brandgrowthos независимо продвигают идею трассировки решений агентов как ключевого инструмента для отладки и улучшения AI-систем. Контекст в 1M токенов делает это практичным. (garrytan)

  5. DeepSeek V4 будет работать на чипах Huawei — Модель адаптируется под Huawei Ascend 950PR (2.87x мощнее Nvidia H20), что станет первым случаем оптимизации крупной модели эксклюзивно для китайского оборудования. (dotey)

  6. Claude Code как операционная система для работы — aakashgupta описал систему трёх папок (knowledge, projects, people), превращающую Claude Code в накопительный рабочий инструмент. К 90-му дню модель самостоятельно выявляет связи между стейкхолдерами и проектами. (aakashgupta)

  7. ChromaFs от Mintlify — виртуальная ФС для AI — Виртуальная файловая система, позволяющая AI работать с документацией через grep/cat/ls. Каждая команда транслируется в запрос к БД, время старта упало с 46 с до 100 мс. Альтернатива классическому RAG. (dotey)

  • Функциональные эмоции LLM — anthropicai, dotey, tom_doerr, onevcat. Исследование Anthropic об эмоциях обсуждается как значимый сигнал для безопасности AI-агентов.

  • Gemma 4 — day-0 поддержка экосистемой — ggerganov, huggingface, ollama, sharbel, atomicbot_ai, atomic_chat_hq. Модель мгновенно появилась в llama.cpp, Ollama, LlamaBarn и локальных клиентах.

  • OpenClaw и его экосистема — steipete, GithubProjects, tom_doerr, garrytan. OpenClaw A2A Gateway, плагины для Claude Code, интеграция с GStack и Ollama Cloud.

  • Decision traces и агентная память — garrytan, brandgrowthos. Логирование решений агентов как ключевой инструмент отладки.

  • Qwen3.5 — от edge до облака — HuggingModels, ollama, roundtablespace. Полная линейка от 0.8B для мобильных до 397B в облаке.
