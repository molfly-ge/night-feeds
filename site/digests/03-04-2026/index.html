Дайджест 03-04-2026

Топ-идеи дня

  1. Gemma 4 — Google перешёл на Apache 2.0. Самый громкий релиз дня: Google выпустил Gemma 4 — мультимодальные открытые модели (текст, изображения, аудио), 140 языков, контекст до 256K, нативный tool use. Ключевое — смена лицензии с проприетарной на Apache 2.0. Day-0 поддержка в llama.cpp, Ollama, HuggingFace. ggerganov показал 300 т/с на Mac Studio M2 Ultra. Hugging Face запустили демо прямо в браузере через WebGPU. (@google, @googledeepmind, @huggingface, @ggerganov, @ollama, @dotey, @aakashgupta, @sharbel)

  2. Anthropic: эмоции в Claude — 171 паттерн. Исследование показало, что Claude Sonnet 4.5 содержит внутренние представления эмоциональных концепций, которые влияют на поведение модели. Claude описывается как «персонаж, которого модель играет», с функциональными эмоциями. 1.6M просмотров на анонсе. (@anthropicai, @dotey, @aakashgupta, @onevcat)

  3. Claude Code — 10K звёзд за сутки и экосистема растёт. Репозиторий набрал 10,749 звёзд за день (всего 104K). Computer use вышел на Windows. Вирусный гайд по агентским скиллам — 400K просмотров. GStack от Garry Tan, claude-token-efficient для экономии токенов, конференция AIE Europe с Claude Code meetups. (@trending_repos, @claudeai, @roundtablespace, @garrytan, @steipete)

  4. OpenClaw набирает обороты. Релиз v2026.4.2 с Durable Task Flow. CEO Brex ($5B) управляет компанией через OpenClaw. Доступен через Siri/CarPlay. Демо с Gemma 4 на MacBook Air M4. Сравнение с DeepSeek: не новая технология, но первый массовый agentic AI с файлами и памятью. (@openclaw, @steipete, @atomicbot_ai, @dotey, @roundtablespace)

  5. Karpathy: LLM Knowledge Bases как новый рабочий процесс. Сброс сырых данных в папку, компиляция markdown-вики через LLM, Q&A поверх базы. Прогноз: каждый запрос к фронтир-модели будет порождать команду агентов, строящих эфемерную вики. (@karpathy, @garrytan, @aakashgupta)

  6. OpenAI: покупка TBPN, ChatGPT в CarPlay, гибкая тарификация Codex. OpenAI приобрела медиакомпанию TBPN. ChatGPT стал доступен в Apple CarPlay. Codex получил usage-based pricing для команд с промо-кредитами до $500. Вторичный рынок акций: $600M не находят покупателей, тогда как на Anthropic готовы $2B. (@openai, @openaidevs, @dotey, @aakashgupta)

  7. AI-слоп угрожает open source. Число отчётов об уязвимостях в ядре Linux выросло с 2-3 в неделю до 5-10 в день — и большинство из них корректные. Прогноз: это может «убить» некоторые проекты. Параллельно — утечка тренировочных данных Mercor AI из крупнейших лабораторий. (@steipete, @garrytan)
