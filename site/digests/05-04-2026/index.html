Топ-идеи дня

  1. Anthropic отключает Claude-подписки от сторонних инструментов — ключевое событие дня. Anthropic прекратила покрытие использования Claude через OpenClaw и другие сторонние интерфейсы. Сообщество разделилось: часть пользователей ищет альтернативы, часть призывает переходить на открытые модели. (dotey, steipete, ollama)

  2. Ollama запускает облако как альтернативу — в ответ на решение Anthropic, Ollama открыла облачный сервис для пользователей OpenClaw: $20/мес, модели GLM-5, Kimi-K2.5, MiniMax-M2.7. (ollama, steipete)

  3. llama.cpp: prompt-based speculative decoding — 300 tok/s — ggerganov продемонстрировал 300 токенов/с на Gemma 4 26B (Mac Studio M2 Ultra) с новой техникой спекулятивной декодировки. (ggerganov)

  4. Взрыв новых моделей на Hugging Face — за сутки появились Qwen3.5-397B-A17B, MiniMax-M2.5, MOSS-TTS, Llama-Guard-4-12B и десятки других. (huggingmodels)

  5. Karpathy строит персональную LLM-вики — Farzapedia: 2500 записей, LLM генерирует статьи о любом аспекте жизни автора. Идея: у каждого агента должен быть свой «файл идей». (karpathy)

  6. Anthropic — 120+ фич за 90 дней — aakashgupta подсчитал темпы релизов Anthropic. Cowork построен агентными роями. Obsidian работает на команде из 3 инженеров при 5M загрузок. (aakashgupta)

  7. DeepSeek V4 будет работать на чипах Huawei Ascend — сигнал об отвязке от NVIDIA в Китае. (dotey)
