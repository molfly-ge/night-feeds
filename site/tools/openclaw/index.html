OpenClaw / Atomic Bot

Тип: Desktop AI Client Сайт: https://atomicbot.ai GitHub: https://github.com/OpenClaw (open source) Источник: @atomicbot_ai, 27-03-2026

AI-клиент с поддержкой локальных моделей через Ollama. Установка в один клик на macOS и Windows.

Модели: MiniMax, Qwen, Kimi и другие через Ollama Режимы: локально (через Ollama) или в облаке Платформа: macOS, Windows (one-click install)
