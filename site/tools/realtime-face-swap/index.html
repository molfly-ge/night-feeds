Realtime Face Swap — Deep-Live-Cam

Тип: Open Source / Computer Vision GitHub: https://github.com/hacksider/Deep-Live-Cam Источник: @roundtablespace, 27-03-2026

Инструмент для реалтайм замены лица через веб-камеру с точным трекингом. Работает локально, 100% open source.

Особенности: - Реалтайм обработка видеопотока с веб-камеры - Точный трекинг лица - Полностью локальный запуск, без отправки данных в облако
