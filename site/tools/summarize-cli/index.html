summarize (CLI)

CLI для локальной суммаризации видео и медиа от Peter Steinberger (steipete).

В версии 0.13 (08-04-2026): - --slides для генерации локальных видео-слайдов - Больше model-backends (включая GitHub Copilot) - Лучшая поддержка GPT-5.4 - Лучшая обработка медиа: HLS detection (m3u8) - Перешёл из личного tap в official homebrew formula → теперь устанавливается через brew install summarize
