Pi (Ollama Coding Agent)

Тип: Coding Agent / Harness Автор: Ollama Twitter: @ollama Источник: @ollama, 30-03-2026

Pi — минималистичный coding agent от Ollama, являющийся движком OpenClaw. Адаптируется под любые рабочие процессы.

Запуск: ollama launch pi --model kimi-k2.5:cloud

Ключевые особенности: - Минимальный harness с мощными базовыми примитивами - Расширяется через extensions, skills, prompt templates, themes - Все Pi packages работают и с Pi, и с Ollama - Поддерживает облачные и локальные модели

Связь: Pi — основа OpenClaw; устанавливается через Ollama.
