AlphaClaw

Тип: Инструмент для быстрого развёртывания OpenClaw
GitHub: https://github.com/chrysb/alphaclaw

Описание

AlphaClaw — упрощённый способ запустить OpenClaw (open-source AI-ассистент, форк Claude) на машине с 8GB RAM. Поддерживает деплой в один клик через Railway или Render прямо из README.

Ключевые возможности

  • Запуск OpenClaw на минимальном железе (8GB RAM)
  • Деплой в один клик через Railway или Render
  • Не требует ручной настройки окружения

Применение

Для всех, кто хочет запустить личный OpenClaw без сложной конфигурации. Рекомендован @garrytan как предпочтительный способ запуска.
