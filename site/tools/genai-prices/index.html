genai-prices

Тип: Python-библиотека / CLI
GitHub: https://github.com/pydantic/genai-prices
Организация: Pydantic

Описание

genai-prices — инструмент от команды Pydantic для расчёта стоимости инференса различных LLM-провайдеров. Позволяет сравнивать цены на токены across OpenAI, Anthropic, Google, Mistral, Cohere и других провайдеров.

Ключевые возможности

  • Актуальная база цен на LLM-инференс по провайдерам
  • Расчёт стоимости для входных и выходных токенов
  • Сравнение провайдеров
  • Интеграция в Python-приложения

Применение

Полезен при оптимизации затрат на LLM в production-системах, для выбора оптимального провайдера под конкретный сценарий использования.
