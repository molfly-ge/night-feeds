19 твитов за сутки. Аккаунт промаутит подборку open-source инструментов от opensourceprojects.dev — главные находки: Oh My codeX (расширения для Codex CLI), PetClaw (локальный desktop-агент-пет), CavemanSpeech (cut 75% выходных токенов), Brain Is Full Crew (Obsidian + AI), Codex/Claude usage-меторы, DNSTT, OpenUsage.

Oh My codeX — расширения для Codex CLI: hooks, agent teams, HUDs, канонические skills, project guidance. Codex остаётся executive-движком, обвязка делает старт сильнее по умолчанию и держит единый workflow от уточнения до завершения. Сайт · Твит

PetClaw — 1-click локальный agent-питомец. Десктопный AI-pet, обходит «configuration tax» — без debug gateway-ошибок, Python-зависимостей и API-ключей. Persistent (живёт на экране, учит привычки), end-to-end автоматизация без manual override. Демо: full outreach pipeline (скрейп креаторов → outbound emails → экспорт в sheet) автономно в фоне. Доступ через бету: invitation code 6VNA7KIR. Beta · Твит

CavemanSpeech — Claude Code/Codex плагин «говорить как пещерный человек». Skill режет ~75% выходных токенов с сохранением технической точности. К нему компаньон, сжимающий memory-файлы — ~45% входных токенов в каждой сессии. Сайт · Твит

Brain Is Full Crew — Obsidian + AI для перегруженных. Не для тех, у кого жизнь под контролем и хочется оптимизации, а для тех, кто тонет и нужна помощь. Сайт · Твит

Codexbar — счётчик потребления Codex/Claude в menu bar. Без логина: per-provider toggles, session+weekly meters с reset countdowns, опциональные Codex web dashboard enrichments (code review remaining, breakdown, credits history), local cost-usage scan последних 30 дней, polling статусов провайдеров с incident-бейджами. Сайт · Твит

OpenUsage — минималистичный дашборд расходов на AI. Живёт в menu bar, показывает в процентах сколько потратил по подпискам кодинг-AI. Сайт · Твит

DNSTT — DNS tunneling, но юзабельный. Враппер вокруг dnstt в multi-platform приложение. Не быстро, но работает, когда всё остальное заблокировано. Страница · Твит
