9 твитов — тезис о том, почему существующая инфраструктура деплоя не готова к AI-агентам.

Агенты деплоят код прямо сейчас — не «скоро». Большинство платформ построены для человека, нажимающего «Deploy» в дашборде. Этот разрыв — архитектурная проблема, не техническая. Твит

Что ломается, когда деплоит агент: 1. Auth — агенты не нажимают кнопки OAuth и не вставляют токены в браузерный UI 2. Isolation — один агент на shared infra влияет на остальных арендаторов 3. API-first — большинство платформ прикрутили API после того, как сделали дашборд; API — гражданин второго сорта

Твит

RailPush строится под этот тезис: 235 MCP tools, API-first с первого дня, полная изоляция на уровне тенанта. railpush.com Твит

MCP важнее, чем кажется. Это не просто «ещё один API-стандарт» — это интерфейсный слой между агентами и всеми инструментами, которые они используют. Твит

Завтрашний флоу деплоя: агент пишет код, тестирует его, деплоит, мониторит и откатывает без единого человека в цикле. Твит
