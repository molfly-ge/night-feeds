Reddit как база данных для аналитики — Обзор reddit-universal-scraper: инструмент для скрапинга любого subreddit или пользователя без API-ключа. Поддерживает извлечение постов, комментариев, медиа, анализ настроений, экспорт в Parquet и интеграцию с Grafana/DuckDB/Metabase. ~419 звёзд на GitHub. Твит | GitHub

Снижение расходов на Claude Code — Проект claude-token-efficient предлагает добавить специальный CLAUDE.md в проект, который подавляет многословие модели. По бенчмаркам — снижение токенов до 17.4%, в отдельных тестах выход сократился на 63%. Твит | GitHub

Flow — AI-ассистент для многоязычной переписки — macOS-инструмент для инди-разработчиков, выходящих на зарубежные рынки. Вызывается клавишей Fn, переводит и генерирует ответы на 100+ языках прямо в окне ввода. Есть память для ссылок и документов. Твит | Сайт

marknative — рендер Markdown в PNG/SVG без браузера — Библиотека со собственным движком компоновки, которая рендерит Markdown напрямую в PNG-буфер без Chromium и DOM. Детерминистический вывод, пакетная обработка, три строки кода. Самый популярный твит аккаунта (~84K просмотров, 723 лайка). Твит | GitHub
