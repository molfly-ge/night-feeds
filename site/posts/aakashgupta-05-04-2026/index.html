48 твитов за сутки. Аккаунт сочетает AI-аналитику с вирусными нарративами из бизнеса, науки и поп-культуры.

Anthropic: 120 фич за 90 дней — Anthropic выпустил 120+ функций за 90 дней: Claude Code, Cowork, пять продуктовых линий одновременно, две новые модели. ARR вырос с $1B до $5B. Aakash сравнил это с "переписыванием правил продуктовой разработки в реальном времени". Твит

Cowork: продукт, построенный агентными роями — четыре инженера построили Cowork за 10 дней, используя swarm-агентов: создание спеки, доски Asana, декомпозиция задач и реализация — всё силами агентов. Сотни агентов, 100 задач. Твит

OpenClaude: публикация 512k строк кода — Anthropic создал внутреннюю систему "Undercover Mode" для защиты внутренней информации. Но затем опубликовал весь исходный код Claude Code (512 000 строк), и кто-то уже выпустил OpenClaude на его основе за 24 часа. Твит

Claude и "вежливые токены" — самая дорогая строка в счёте за Claude — это слово "пожалуйста". Автор цитирует исследование: "I'd be happy to help you with that!" стоит 9 токенов. Промпт типа "пещерного человека" снижает расход токенов на 75%. Твит

Claude и лимиты сессий — Anthropic ужесточил лимиты в часы пик 26 марта. Пользователь сообщил: слово "Hello" потребило 13% всей сессии. Aakash разбирает почему: системный промпт + история контекста составляют большую часть счётчика токенов. Твит

Karpathy и LLM-вики без кода — Андрей Карпати "распространил продукт без единой строки кода": одна страница Markdown с описанием, как строить LLM-поддерживаемую личную вики. Сырые документы в папку, LLM "переваривает" их и создаёт базу знаний. Твит

Claude Code как персональная CRM — Carl Vellotti ведёт папку "people" в Claude Code workspace: досье на каждого коллегу — приоритеты, позиции из встреч, интересы. Твит

Perplexity: ставка на агрегацию моделей — год назад 90% запросов Perplexity шло к двум моделям. Сегодня ни одна модель не занимает >25% трафика. Они не строили ни одну из 19 запущенных моделей — это и есть их тезис. Твит

Bolt vs Lovable: раскол рынка AI-прототипирования — Bolt: $700M оценка, $40M ARR, 20 человек, работает в браузере на WebContainers (7 лет разработки). Lovable: $6.6B оценка. Два принципиально разных подхода к одному рынку. Твит

Obsidian: три инженера, 5M загрузок — команда из 3 инженеров создала продукт с 5M загрузками, 1.5M активных пользователей, 2000+ плагинов от сообщества. Они нанимают четвёртого. Твит

Microsoft Copilot: худший квартал с 2008 — акции Microsoft упали на 23% за квартал (хуже всех в Mag 7). Надела лично курировал Copilot. Проблема: ИИ не может модифицировать документ Word прямо из интерфейса Word. Твит
