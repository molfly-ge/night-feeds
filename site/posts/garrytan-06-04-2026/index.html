40 твитов за сутки. Гарри Тан в режиме берсерка — выкатил GStack для OpenClaw, строит GBrain на SQLite, анонсирует браузерного агента и продвигает опенсорс-философию «программирование для всех».

GStack для OpenClaw — официальный релиз — GStack-lite теперь интегрирован с OpenClaw. Когда OpenClaw использует Claude Code (а он это делает постоянно), GStack добавляет «крылья» — задачи выполняются быстрее и умнее. 14 секьюрити-фиксов в свежем релизе, половина от комьюнити. Твит

GBrain — переход с git-wiki на SQLite — Karpathy-style база знаний для OpenClaw доросла до 2.3 ГБ, а лимит git — 5 ГБ. GStack autoplan сгенерировал спеку для новой версии GBrain на SQLite с миграцией на Postgres/pgvector в будущем. Будет опенсорс под MIT. Твит | Gist со спекой

GStack Browser — превью — OpenClaw скоро сможет управлять браузером на вашем компьютере через GStack Browser. Пока снимок экрана, но выглядит рабочим. Твит
