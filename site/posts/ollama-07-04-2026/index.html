6 твитов за сутки. Ollama объявил о запуске Gemma 4 в облаке на GPU NVIDIA Blackwell и рассказал о масштабной совместной работе по подготовке релиза.

Gemma 4 в Ollama Cloud на GPU Blackwell — Облако Ollama теперь является лучшим местом для запуска Gemma 4: доступно по подписке для разработчиков и интеграций третьих сторон, работает на последних GPU NVIDIA Blackwell. (Твит, Gemma 4 на Ollama)

Межкомандная интеграция для Gemma 4 — До релиза Ollama провёл совместную работу с HuggingFace, vLLM и командами внутри Google для обеспечения совместимости модели. (Твит)

Пользовательское приложение для eBay-бизнеса — Один из пользователей поделился приложением для отслеживания товаров на eBay с AI-функциями для малого бизнеса, собранным без API-ключей с локальными моделями. (Твит)
