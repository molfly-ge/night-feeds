4 твита за сутки. Обзор инструментов для безопасности сайтов и AI-разработки дизайна.

Shannon — инструмент автоматического пентеста — axiaisacat рассказал об инструменте Shannon (36k звёзд на GitHub), который за одну команду (npx [@keygraph](https://x.com/keygraph?s=11)/shannon) автоматически проверяет сайт на SQL-инъекции, XSS, SSRF, проблемы с JWT и обходы аутентификации. Позиционируется как способ проверить свой сайт до того, как это сделают злоумышленники. Твит, GitHub

awesome-design-md — дизайн-документы для AI — открытый проект, который собирает дизайн-языки топовых продуктов (Claude, Linear, Vercel, Stripe) в формате DESIGN.md, пригодном для чтения AI-агентами. Позволяет передать AI точные правила по цветам, шрифтам, отступам, компонентам и иерархии перед генерацией страниц. Твит, GitHub
