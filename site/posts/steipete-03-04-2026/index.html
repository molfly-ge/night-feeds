6 твитов за сутки.

AI-слоп захлёстывает Linux-ядро — по словам Steinberger, на списке рассылки kernel security число отчётов об уязвимостях выросло с 2-3 в неделю до 5-10 в день, причём большинство из них корректные. Прогноз: это может «убить» некоторые open-source проекты. Твит

Plan mode не нужен — Steinberger утверждает, что никогда не использует plan mode в Codex/Claude Code. По его мнению, эта функция добавлена для людей, привыкших к Claude, которым сложно менять привычки. Лучше просто разговаривать с агентом. Твит

OpenClaw через Siri и CarPlay — ретвит о том, что Telegram iOS теперь позволяет общаться с ботами OpenClaw через Siri/CarPlay: достаточно создать контакт с Telegram-ссылкой на бота. Твит оригинала

CEO Brex управляет компанией ($5 млрд) через OpenClaw — ретвит с подробностями от CEO Brex о реальном использовании агентов в управлении крупной компанией. Твит оригинала

OpenAI Codex как шутка — ретвит с ироничным советом «решить» проблему лимитов Claude Code установкой [@openai](https://x.com/openai?s=11)/codex. Твит оригинала
