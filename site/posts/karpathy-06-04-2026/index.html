3 твита за сутки. Карпати обсуждает X API, ценообразование для AI-агентов и хвалит комментарии на GitHub Gists.

X API и XMCP — дорого и непонятно — Карпати попробовал использовать X API для проекта и за 30 минут потратил $200. Ценообразование, по его мнению, чрезмерное. Документация сложна для загрузки в агенты (много коротких страниц вместо одного большого markdown). Также не нашёл упоминания XMCP в документации. В продолжении темы предложил сделать Read-эндпоинты дешевле, а Write — дороже, чтобы бороться с AI-спамом. Твит

GitHub Gists как альтернатива X — Удивился качеству комментариев на GitHub Gists: более полезные, конструктивные и с меньшим количеством AI-сгенерированного контента. Предположил, что дело в сообществе, формате Markdown или отсутствии стимулов для спама. Призвал GitHub конкурировать с X. Твит
