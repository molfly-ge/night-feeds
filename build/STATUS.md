# Статус проекта

## Сделано

### Пайплайн
- `run-digest.sh` — запуск через `claude --dangerously-skip-permissions`, без подтверждений
- Cron на 8:00 ежедневно на машине сборки
- `CLAUDE.md` обновлён: извлекать URLs из твитов, добавлять раздел «Источники» в посты, карточки для известных продуктов с новыми релизами

### Статический сайт (`build/build.py`)
- Генерация 57 страниц: `/`, `/digests/{date}`, `/posts/{account}-{date}`, `/tools/{name}`
- Чистые URL без расширений (каждая страница — директория с index.html)
- @mentions → ссылки на x.com
- `**name** (github-user)` → ссылка на github.com/user/name
- Имена инструментов в тексте → автолинк на `/tools/{slug}`
- Голые `github.com/user/repo` → кликабельные ссылки
- YAML frontmatter стрипается
- Строка «Обработано:» убрана
- PNG-карточки встраиваются в посты и страницы инструментов

### Деплой
- `deploy.sh` — `git pull` → `python3 build/build.py` → `rsync site/ → /var/www/rss.molfly.me/`
- Cron на 9:00 на сервере деплоя
- Caddyfile с блоком `rss.molfly.me`

## Ждёт проверки завтра

- Посты с разделом «Источники» и inline-ссылками из твитов (первый запуск по новому CLAUDE.md)
- Полный цикл: сборка в 8:00 → деплой в 9:00 → сайт обновлён
