# Дайджест 06-04-2026

## Топ-идеи дня

1. **Anthropic продолжает блокировать OpenClaw — steipete переключается на GPT** — Anthropic блокирует OpenClaw даже через собственный CLI `claude -p`, фильтруя по ключевому слову в system prompt. Steipete убрал фичу Claude-подписки из OpenClaw и заявил, что GPT 5.4 «стабильнее». 4200+ лайков, 1M+ просмотров. ([steipete](https://x.com/steipete/status/2040811558427648357), [dotey](https://x.com/dotey/status/2040854097838952598))

2. **Goose от Block — бесплатный AI-кодинг-агент с 35k звёздами** — Jack Dorsey's Block выложил в open source агента, который работает с любой LLM (Claude, GPT, Gemini, Llama). Rust, три интерфейса (десктоп, CLI, веб). BrandGrowthOS сравнил с Claude Code: навигация хуже, но «$180/месяц — это $180/месяц». ([roundtablespace](https://x.com/RoundtableSpace/status/2040780242411737438), [brandgrowthos](https://x.com/BrandGrowthOS/status/2040778485300674931))

3. **Gemma 4 делает AI-агентов бесплатными** — Google выпустил Gemma 4, она работает на iPhone 17 Pro (40 tok/s), MacBook и оркестрирует другие модели для видео-сегментации. Главное: OpenClaw + Gemma 4 = агенты без API-расходов. ([roundtablespace](https://x.com/RoundtableSpace/status/2040976535469912572), [roundtablespace](https://x.com/RoundtableSpace/status/2040840640250937796))

4. **Alibaba собирает данные разработчиков через бесплатные токены** — Qwen 3.6 Plus обработал 1.4 трлн токенов за сутки на OpenRouter. $0 за вход и выход. Китайские модели выросли с 2% до 45% трафика OpenRouter за год. Каждый промпт уходит на файн-тюнинг. ([aakashgupta](https://x.com/aakashgupta/status/2040874117881286754))

5. **От автономных агентов к автономным воркфлоу** — BrandGrowthOS описал путь: от ручной координации агентов до полностью автономного контент-пайплайна через n8n-вебхуки. Агенты передают контекст напрямую и дебажат друг друга. ([brandgrowthos](https://x.com/BrandGrowthOS/status/2040867688608870456))

6. **Karpathy: X API слишком дорог для агентов** — $200 за 30 минут. Предложил сделать Read-эндпоинты дешевле, Write — дороже для борьбы с AI-спамом. Отдельно хвалит GitHub Gists за качество комментариев. ([karpathy](https://x.com/karpathy/status/2040838208674734473))

7. **Perplexity Computer — консалтинг за $20/мес** — 400+ коннекторов к CRM, Snowflake, FactSet. Сравнили с McKinsey по 16000 запросам. 100 enterprise-клиентов за первые выходные. ([aakashgupta](https://x.com/aakashgupta/status/2040844170999517597))

## Пересечения тем

- **Anthropic vs OpenClaw (день 2)** — steipete, dotey, roundtablespace, aakashgupta независимо обсудили блокировку. Steipete переходит на GPT, dotey анализирует экономику токенов, roundtablespace сравнивает Claude Cowork и OpenClaw. Консенсус: ограничения Anthropic подталкивают экосистему к мультимодельности.

- **Локальные модели как ответ на rate limits** — ollama (Gemma 4, Qwen 3.5), brandgrowthos (домашний сервер с 4090), roundtablespace (Gemma 4 на iPhone). Причина одна: облачные API дорожают и ограничиваются, локальный инференс становится практичным.

- **Компания одного человека** — roundtablespace (1113 лайков: «один основатель, шесть AI-отделов, ноль сотрудников»), brandgrowthos (6 агентов на домашнем сервере), garrytan («самый крутой момент для людей с высокой агентностью»). Нарратив «AI заменяет команды» набирает силу.

- **Базы знаний важнее кода** — brandgrowthos (Obsidian + Claude для организации данных), aakashgupta (Karpathy: главный навык — управление агентами, не код), garrytan (GBrain на SQLite для OpenClaw). Тренд: от code generation к knowledge management.
