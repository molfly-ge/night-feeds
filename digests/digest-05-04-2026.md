# Дайджест 05-04-2026

## Топ-идеи дня

1. **Anthropic отключает Claude-подписки от сторонних инструментов** — ключевое событие дня. Anthropic прекратила покрытие использования Claude через OpenClaw и другие сторонние интерфейсы. Сообщество разделилось: часть пользователей ищет альтернативы, часть призывает переходить на открытые модели. (dotey, steipete, huggingface, aakashgupta, ollama)

2. **Ollama запускает облако как альтернативу** — в ответ на решение Anthropic, Ollama открыла облачный сервис для пользователей OpenClaw: $20/мес, модели GLM-5, Kimi-K2.5, MiniMax-M2.7. (ollama, steipete)

3. **llama.cpp: prompt-based speculative decoding — 300 tok/s** — ggerganov продемонстрировал 300 токенов/с на Gemma 4 26B (Mac Studio M2 Ultra) с новой техникой спекулятивной декодировки. (ggerganov)

4. **Взрыв новых моделей на Hugging Face** — за сутки появились Qwen3.5-397B-A17B, MiniMax-M2.5, MOSS-TTS, Llama-Guard-4-12B и десятки других. (huggingmodels)

5. **Karpathy строит персональную LLM-вики** — Farzapedia: 2500 записей, LLM генерирует статьи о любом аспекте жизни автора. Идея: у каждого агента должен быть свой «файл идей». (karpathy)

6. **Anthropic — 120+ фич за 90 дней** — aakashgupta подсчитал темпы релизов Anthropic. Cowork построен агентными роями. Obsidian работает на команде из 3 инженеров при 5M загрузок. (aakashgupta)

7. **DeepSeek V4 будет работать на чипах Huawei Ascend** — сигнал об отвязке от NVIDIA в Китае. (dotey)

## Пересечения тем

- **Отключение Claude от сторонних инструментов** — независимо обсудили dotey, steipete, huggingface, aakashgupta, ollama, sharbel. Это главная тема дня с разных ракурсов: бизнес-решение, реакция сообщества, технические альтернативы.

- **Открытые модели как альтернатива закрытым** — huggingface призывает переходить на открытые модели, ollama предлагает облако, sharbel переключается на Gemma 4 и OpenAI Codex. Тренд на диверсификацию от одного провайдера.

- **Claude Code как платформа** — garrytan (GStack Browse), steipete (кастомизация агента), roundtablespace (торговые боты через Claude Code). Инструмент становится экосистемой.

- **Speculative decoding и скорость инференса** — ggerganov (llama.cpp 300 tok/s), huggingmodels (новые модели с MoE-архитектурами). Скорость локального инференса растёт.
