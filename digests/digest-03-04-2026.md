# Дайджест 03-04-2026

## Топ-идеи дня

1. **Gemma 4 — Google перешёл на Apache 2.0.** Самый громкий релиз дня: Google выпустил Gemma 4 — мультимодальные открытые модели (текст, изображения, аудио), 140 языков, контекст до 256K, нативный tool use. Ключевое — смена лицензии с проприетарной на Apache 2.0. Day-0 поддержка в llama.cpp, Ollama, HuggingFace. ggerganov показал 300 т/с на Mac Studio M2 Ultra. Hugging Face запустили демо прямо в браузере через WebGPU. ([@google](posts/post-google-03-04-2026.md), [@googledeepmind](posts/post-googledeepmind-03-04-2026.md), [@huggingface](posts/post-huggingface-03-04-2026.md), [@ggerganov](posts/post-ggerganov-03-04-2026.md), [@ollama](posts/post-ollama-03-04-2026.md), [@dotey](posts/post-dotey-03-04-2026.md), [@aakashgupta](posts/post-aakashgupta-03-04-2026.md), [@sharbel](posts/post-sharbel-03-04-2026.md))

2. **Anthropic: эмоции в Claude — 171 паттерн.** Исследование показало, что Claude Sonnet 4.5 содержит внутренние представления эмоциональных концепций, которые влияют на поведение модели. Claude описывается как «персонаж, которого модель играет», с функциональными эмоциями. 1.6M просмотров на анонсе. ([@anthropicai](posts/post-anthropicai-03-04-2026.md), [@dotey](posts/post-dotey-03-04-2026.md), [@aakashgupta](posts/post-aakashgupta-03-04-2026.md), [@onevcat](posts/post-onevcat-03-04-2026.md))

3. **Claude Code — 10K звёзд за сутки и экосистема растёт.** Репозиторий набрал 10,749 звёзд за день (всего 104K). Computer use вышел на Windows. Вирусный гайд по агентским скиллам — 400K просмотров. GStack от Garry Tan, claude-token-efficient для экономии токенов, конференция AIE Europe с Claude Code meetups. ([@trending_repos](posts/post-trending_repos-03-04-2026.md), [@claudeai](posts/post-claudeai-03-04-2026.md), [@roundtablespace](posts/post-roundtablespace-03-04-2026.md), [@garrytan](posts/post-garrytan-03-04-2026.md), [@steipete](posts/post-steipete-03-04-2026.md))

4. **OpenClaw набирает обороты.** Релиз v2026.4.2 с Durable Task Flow. CEO Brex ($5B) управляет компанией через OpenClaw. Доступен через Siri/CarPlay. Демо с Gemma 4 на MacBook Air M4. Сравнение с DeepSeek: не новая технология, но первый массовый agentic AI с файлами и памятью. ([@openclaw](posts/post-openclaw-03-04-2026.md), [@steipete](posts/post-steipete-03-04-2026.md), [@atomicbot_ai](posts/post-atomicbot_ai-03-04-2026.md), [@dotey](posts/post-dotey-03-04-2026.md), [@roundtablespace](posts/post-roundtablespace-03-04-2026.md))

5. **Karpathy: LLM Knowledge Bases как новый рабочий процесс.** Сброс сырых данных в папку, компиляция markdown-вики через LLM, Q&A поверх базы. Прогноз: каждый запрос к фронтир-модели будет порождать команду агентов, строящих эфемерную вики. ([@karpathy](posts/post-karpathy-03-04-2026.md), [@garrytan](posts/post-garrytan-03-04-2026.md), [@aakashgupta](posts/post-aakashgupta-03-04-2026.md))

6. **OpenAI: покупка TBPN, ChatGPT в CarPlay, гибкая тарификация Codex.** OpenAI приобрела медиакомпанию TBPN. ChatGPT стал доступен в Apple CarPlay. Codex получил usage-based pricing для команд с промо-кредитами до $500. Вторичный рынок акций: $600M не находят покупателей, тогда как на Anthropic готовы $2B. ([@openai](posts/post-openai-03-04-2026.md), [@openaidevs](posts/post-openaidevs-03-04-2026.md), [@dotey](posts/post-dotey-03-04-2026.md), [@aakashgupta](posts/post-aakashgupta-03-04-2026.md))

7. **AI-слоп угрожает open source.** Число отчётов об уязвимостях в ядре Linux выросло с 2-3 в неделю до 5-10 в день — и большинство из них корректные. Прогноз: это может «убить» некоторые проекты. Параллельно — утечка тренировочных данных Mercor AI из крупнейших лабораторий. ([@steipete](posts/post-steipete-03-04-2026.md), [@garrytan](posts/post-garrytan-03-04-2026.md))

## Пересечения тем

- **Gemma 4 и локальный запуск** — 10+ аккаунтов независимо обсудили: @google, @googledeepmind, @huggingface, @ggerganov, @ollama, @dotey, @aakashgupta, @sharbel, @atomicbot_ai, @atomic_chat_hq, @roundtablespace. - **Claude Code / агентные инструменты** — @roundtablespace, @garrytan, @steipete, @sharbel, @axiaisacat, @trending_repos, @claudeai. Темы: skills management через Git, экономия токенов, computer use на Windows, 10K звёзд за сутки.

- **OpenClaw как массовый agentic AI** — @openclaw, @steipete, @atomicbot_ai, @atomic_chat_hq, @dotey, @roundtablespace, @sharbel, @tom_doerr. Релиз v2026.4.2, интеграция с Siri/CarPlay, запуск с Gemma 4 на MacBook, sandbox-альтернатива poco-claw.

- **OpenAI: продуктовые ходы и рынок** — @openai, @openaidevs, @openainewsroom, @aakashgupta, @dotey, @garrytan. Покупка TBPN, CarPlay, Codex pricing, вторичный рынок акций ($600M без покупателей).

- **Открытые модели и файнтюны** — @huggingface, @huggingmodels, @garrytan. Qwopus 27B v3 (дистилляция из Claude 4.6 Opus), Qwen3.5 серия, abliterated-версии. Экосистема открытых моделей расширяется быстрее закрытых.

- **Perplexity Computer для налогов** — @perplexity_ai и @roundtablespace: AI-агент заполняет федеральную налоговую декларацию.
