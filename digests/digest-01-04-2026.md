# Дайджест 01-04-2026

## Топ-идеи дня

**1. Утечка исходного кода Claude Code — случайный open source.**
Anthropic забыл добавить `*.map` в `.npmignore`. Bun (который принадлежит Anthropic) генерирует source maps по умолчанию. Результат: 512 000 строк исходного кода в npm-пакете на публичном Cloudflare bucket. 41 500 форков на GitHub за несколько часов. Внутри: 44 unreleased feature flag, anti-distillation механизмы (фальшивые tool calls, скрытые reasoning chains, изолированный JSON-протокол), dream-движок консолидации памяти, Tamagotchi-компаньон `/buddy` (запланирован на 1 апреля). И 29–30% false claims rate в последней версии модели против 16.7% в v4.
— [@roundtablespace](posts/post-roundtablespace-01-04-2026.md), [@aakashgupta](posts/post-aakashgupta-01-04-2026.md), [@axiaisacat](posts/post-axiaisacat-01-04-2026.md), [@garrytan](posts/post-garrytan-01-04-2026.md), [@dotey](posts/post-dotey-01-04-2026.md), [@GithubProjects](posts/post-GithubProjects-01-04-2026.md)

**2. Ollama 0.19: MLX-бэкенд на Apple Silicon.**
Ollama теперь использует MLX для ускорения на Mac — 2–3x прирост скорости генерации. На M5 через GPU Neural Accelerators. NVFP4 для NVIDIA. Умное кэширование промптов. Стартовая модель: Qwen3.5-35B-A3B-nvfp4 (нужно >32 GB).
— [@ollama](posts/post-ollama-01-04-2026.md), [@roundtablespace](posts/post-roundtablespace-01-04-2026.md)

**3. OpenAI $122B — деньги циркулируют, не поступают.**
Amazon вложил $50B и расширил AWS-коммитмент OpenAI на $100B. Nvidia вложила $30B и продаёт OpenAI GPU. SoftBank вложил $30B через мостовой кредит $40B. Оценка $852B при 35x revenue multiple у компании, никогда не получавшей прибыль, сжигающей $150M в день. Runway: 18–24 месяца.
— [@aakashgupta](posts/post-aakashgupta-01-04-2026.md), [@roundtablespace](posts/post-roundtablespace-01-04-2026.md)

**4. Китайская платформенная война за OpenClaw за одну неделю.**
OpenClaw (310K GitHub stars) спровоцировал ответ четырёх триллионно-долларовых компаний: ArkClaw (ByteDance), WorkBuddy (Tencent), QoderWork (Alibaba), AutoClaw (Zhipu). Zhipu: no API key, local model, 50 pre-built skills. Акции +13%. Шэньчжэнь платит разработчикам до 2 млн юаней.
— [@aakashgupta](posts/post-aakashgupta-01-04-2026.md)

**5. npm axios supply chain attack.**
Популярнейшая HTTP-библиотека JavaScript (300M загрузок/неделю): версии axios@1.14.1 и axios@0.30.4 содержат Remote Access Trojan. Не первый такой инцидент в npm.
— [@karpathy](posts/post-karpathy-01-04-2026.md), [@dotey](posts/post-dotey-01-04-2026.md)

**6. Perplexity — Stripe для AI.**
Оценка $20B, не обучила ни одного параметра. Маршрутизирует к 19 моделям. В январе 2025 два модели занимали 90% трафика, к декабрю — ни одна не выше 25%. ARR ~$200M. Четыре компании Mag Seven используют их Search API в production и ничего не могут сделать: отключить = потерять дистрибуцию.
— [@aakashgupta](posts/post-aakashgupta-01-04-2026.md)

**7. TurboQuant/TQ3_1S: 27B модель в 12.9 GB на RTX 5060 Ti.**
Google делал для KV cache compression — сообщество применило к весам. 0.19% PPL gap от Q4_0, 1.5 GB экономии, которые решают войдёт ли модель в 16 GB VRAM. 96.91% HumanEval, -24% chain-of-thought bloat.
— [@huggingface](posts/post-huggingface-01-04-2026.md), [@aakashgupta](posts/post-aakashgupta-01-04-2026.md)
