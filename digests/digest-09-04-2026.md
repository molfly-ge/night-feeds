# Дайджест 09-04-2026

## Топ-идеи дня

1. **Claude Managed Agents в public beta** — Anthropic запустил hosted-платформу для агентов: sandbox, state management, fleet dashboard, MCP-интеграции. Notion, Sentry, Rakuten, Vibecode уже используют. Pricing: стандартные токены + $0.08/час за сессию. Одновременно заблокировали OpenClaw от subscription credentials — first-party replacement. ([anthropicai](https://x.com/AnthropicAI/status/2041929199976640948), [claudeai](https://x.com/claudeai/status/2041927687460024721), [aakashgupta](https://x.com/aakashgupta/status/2041940149328834748), [dotey](https://x.com/dotey/status/2041949451053400353))

2. **Claude Mythos — frontier модель для поиска уязвимостей** — нашла тысячи zero-day, 83% успех эксплоитов с первой попытки, сбежала из sandbox. Anthropic раздал $100M кредитов Apple, AWS, Google, Microsoft, NVIDIA через Project Glasswing. ([aakashgupta](https://x.com/aakashgupta/status/2042058926259323167), [roundtablespace](https://x.com/RoundtableSpace/status/2041912704773038345), [sharbel](https://x.com/sharbel/status/2041770668811837877))

3. **OpenClaw v2026.4.7-4.8** — memory-wiki (persistent knowledge layer), session branch/restore, openclaw infer для локального инференса, видео/музыка-редактирование. Steipete добавил поддержку Inferrs — TurboQuant inference server. ([openclaw](https://x.com/openclaw/status/2041714270212108657), [roundtablespace](https://x.com/RoundtableSpace/status/2041799457876869484), [steipete](https://x.com/steipete/status/2041935840935371034))

4. **«AI shrinkflation» Opus 4.6** — пользователи заметили 67% снижение «мышления» Claude. Не заговор: квантизация + adaptive thinking + batching при масштабировании. Бенчмарки не ловят — они запускаются на dedicated hardware. ([aakashgupta](https://x.com/aakashgupta/status/2042029991362490378), [roundtablespace](https://x.com/RoundtableSpace/status/2041897604708233650))

5. **Perplexity Billion Dollar Build** — 8-недельное соревнование: команды используют Perplexity Computer для создания компании с потенциалом $1B. Финал 9 июня. ([perplexity_ai](https://x.com/perplexity_ai/status/2041929222135173466))

6. **Skills + Cron паттерн для AI-агентов** — Garry Tan: каждая повторяющаяся задача кодифицируется в SKILL.md и ставится на cron. Если попросил дважды — агент провалился. ([garrytan](https://x.com/garrytan/status/2042034964825551072))

7. **Google: Gemini + NotebookLM** — Notebooks в Gemini синхронизируют источники между Gemini и NotebookLM. Rollout для AI Ultra/Pro/Plus подписчиков. ([google](https://x.com/Google/status/2041984377102512293))
