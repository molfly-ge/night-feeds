# Дайджест 31-03-2026

## Топ-идеи дня

1. **Claude Code computer use замкнул петлю разработки** — 17 месяцев от Docker-демо к CLI: теперь Claude пишет код, открывает приложение, кликает по UI, находит баг и исправляет — в одном сеансе терминала. Только macOS, Pro/Max. ([aakashgupta](posts/post-aakashgupta-31-03-2026.md), [dotey](posts/post-dotey-31-03-2026.md), [roundtablespace](posts/post-roundtablespace-31-03-2026.md))

2. **OpenAI официально вошла в экосистему Claude Code** — плагин codex-plugin-cc: `/codex:adversarial-review` ловит баги, которые Claude пропускает в своём собственном коде. Подтверждено экспериментом: Claude написал игру, пропустил soft-lock баг, Codex поймал с первого прохода. ([axiaisacat](posts/post-axiaisacat-31-03-2026.md), [dotey](posts/post-dotey-31-03-2026.md), [brandgrowthos](posts/post-brandgrowthos-31-03-2026.md))

3. **RAM-кризис и TurboQuant** — самая резкая разворачивающаяся история: Sam Altman подписал 40% мирового предложения DRAM, Micron убила Crucial, потом Google выпускает алгоритм 6× сжатия. Micron -33% при выручке +196%. ([aakashgupta](posts/post-aakashgupta-31-03-2026.md))

4. **CLI > MCP > API** — иерархия подтверждена Karpathy и подробно разобрана: MCP-серверы занимают контекст постоянно, CLIs — только по вызову. Лучшие пользователи Claude Code удалили большинство MCP. ([aakashgupta](posts/post-aakashgupta-31-03-2026.md), [dotey](posts/post-dotey-31-03-2026.md))

5. **llama.cpp достиг 100K звёзд** — Georgi Gerganov прогнозирует, что 90% AI-агентов будут работать локально с llama.cpp через 3-6 месяцев. Local agentic workflows — реальность с лета 2025. ([huggingface](posts/post-huggingface-31-03-2026.md))

6. **Eval постоянен, промпты — нет** — каждый swap модели обнуляет промпты, но не eval. Компании, инвестирующие в prompt engineering вместо eval, строят на песке. ([aakashgupta](posts/post-aakashgupta-31-03-2026.md))

7. **Starcloud — быстрейший единорог YC** — $170M Series A при $1.1B оценке. Уже обучила AI в космосе на H100. EQT (70+ датацентров) инвестирует в орбитальные датацентры — сигнал что земля заканчивается. ([aakashgupta](posts/post-aakashgupta-31-03-2026.md), [garrytan](posts/post-garrytan-31-03-2026.md))
