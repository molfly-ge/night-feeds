# Дайджест 29-03-2026

## Топ-идеи дня

1. **Инженер Anthropic = менеджер агентов** (@RoundtableSpace, 721 лайк) — Anthropic-инженеры всё меньше пишут код и всё больше управляют несколькими агентами параллельно. Смена парадигмы зафиксирована изнутри компании.

2. **Demo работает, production ломается** (@BrandGrowthOS) — Главная причина: AI-демо создаются на чистых предсказуемых данных, реальный production — грязные, непоследовательные данные с меняющимися форматами. Проблема не в модели, а в недотестированных edge-cases.

3. **n8n 2.14.0 beta: MCP внутри** (@BrandGrowthOS) — Теперь AI-агент может создавать и обновлять n8n-воркфлоу прямо через MCP из Claude Code/Cursor. Агенты, строящие собственные automation-пайплайны — реальность.

4. **Claude Code навыки для видео** (@RoundtableSpace) — Появился claude-code-video-toolkit: видеопроизводство прямо из Claude Code. Экосистема навыков продолжает расширяться.

5. **Google разграничил: AI-агент ≠ краулер** (@BrandGrowthOS) — Официальное разъяснение Google: агентский веб-доступ (user-triggered) и GoogleBot — разные режимы с разными правилами. Многие production-системы были в серой зоне.

6. **AI-Scientist-v2 генерирует статьи** (@tom_doerr) — SakanaAI выпустила вторую версию системы для автономной генерации научных статей через агентный tree search.

7. **homelable — Infrastructure visualizer** (@tom_doerr, 174 лайка) — Self-hosted визуализатор инфраструктуры со сканером сети стал одним из самых популярных репозиториев дня.
