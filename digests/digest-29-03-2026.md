# Дайджест 29-03-2026

## Топ-идеи дня

1. **Инженер Anthropic = менеджер агентов** (@RoundtableSpace, 721 лайк) — Anthropic-инженеры всё меньше пишут код и всё больше управляют несколькими агентами параллельно. Смена парадигмы зафиксирована изнутри компании.

2. **Demo работает, production ломается** (@BrandGrowthOS) — Главная причина: AI-демо создаются на чистых предсказуемых данных, реальный production — грязные, непоследовательные данные с меняющимися форматами. Проблема не в модели, а в недотестированных edge-cases.

3. **n8n 2.14.0 beta: MCP внутри** (@BrandGrowthOS) — Теперь AI-агент может создавать и обновлять n8n-воркфлоу прямо через MCP из Claude Code/Cursor. Агенты, строящие собственные automation-пайплайны — реальность.

4. **Claude находит zero-day уязвимости в реальных системах** (@garrytan, 8358 лайков) — Anthropic показал на конференции live-демо поиска zero-day через Claude. Garry Tan уже использует `/design-consultation` и называет это точкой невозврата.

5. **AEO — Agent Engine Optimization** (@sharbel) — Новая концепция: пользователи делегируют поиск AI-агентам, и «ранжирование» теперь означает цитирование агентами, а не позицию в Google. Большинство создателей контента ещё не осознали сдвиг.

6. **TRIBE v2 — цифровой двойник мозга** (@sharbel) — Meta AI представила модель на 500+ часах fMRI от 700+ людей. Предсказывает нейронные реакции zero-shot. AI начинает понимать, как мы воспринимаем реальность.

7. **AI-Scientist-v2 генерирует статьи** (@tom_doerr) — SakanaAI выпустила вторую версию системы для автономной генерации научных статей через агентный tree search.

8. **GStack** (@garrytan) — Навыки для Claude Code от CEO YC: `/design-consultation`, `/codex`, `/tasks`. Принцип User Sovereignty: AI рекомендует, человек решает.

9. **GitHub Copilot CLI генерирует unit-тесты** (@github) — Workflow: plan mode → флот агентов → мониторинг через `/tasks`. Прямо из терминала.

10. **homelable — Infrastructure visualizer** (@tom_doerr, 174 лайка) — Self-hosted визуализатор инфраструктуры со сканером сети стал одним из самых популярных репозиториев дня.
