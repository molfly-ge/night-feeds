# Project Glasswing

Инициатива Anthropic по защите критически важного программного обеспечения. Программа позиционируется как «urgent» — массовый поиск уязвимостей в инфраструктуре до того, как их найдут атакующие.

Движок программы — новая фронтир-модель **Claude Mythos Preview**. Модель не выпускается публично: вместо этого Anthropic ограниченно отдала её 12 крупным вендорам (Apple, Microsoft, Amazon и др.) специально для cyber-defenders.

**Уже найденное:**
- 27-летняя уязвимость в OpenBSD: достаточно отправить несколько пакетов на любой OpenBSD-сервер, чтобы он упал
- Несколько багов в Linux, позволяющих непривилегированному пользователю поднять привилегии до root, просто запустив бинарник
