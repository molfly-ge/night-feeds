# TopRank

Open-source Python-инструмент для SEO-анализа конкурентов. Позволяет сравнивать позиции сайтов, анализировать ключевые слова и отслеживать изменения в поисковой выдаче. Ориентирован на маркетологов и SEO-специалистов, предпочитающих программные решения облачным сервисам.
