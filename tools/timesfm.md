# TimesFM (Google Research)

**Опенсорс AI-модель для прогнозирования временных рядов от Google Research.**

Foundation model для time-series forecasting: предсказывает что угодно — продажи, трафик, погоду, финансовые данные. Претренирована на огромном корпусе временных рядов, работает без дообучения на новых доменах.
