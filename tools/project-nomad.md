# Project Nomad

Офлайн-сервер знаний и образования — устанавливается одной командой, работает без интернета.

**Возможности:** локальный AI-чат с загрузкой документов и семантическим поиском, офлайн Wikipedia, медсправочники, электронные книги, курсы Khan Academy с отслеживанием прогресса, скачиваемые карты, инструменты шифрования, локальные заметки.
