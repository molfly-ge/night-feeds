# Poco Claw

**Безопасная песочница — альтернатива OpenClaw** для изолированного запуска AI-агентов.

Poco Claw — инструмент для запуска AI-агентов в изолированной песочнице. Позиционируется как более безопасная альтернатива OpenClaw с улучшенной изоляцией и контролем доступа.
