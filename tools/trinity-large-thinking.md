# Trinity-Large-Thinking (Arcee AI)

**Открытая reasoning-модель от Arcee AI с открытыми весами на HuggingFace.**

Новая модель с «thinking» (цепочкой рассуждений), доступна через Arcee API и как открытые веса. Вышла 1 апреля 2026 — без первоапрельских шуток. Arcee специализируется на domain-specific LLM для enterprise.
