# Harrier

Open-source embedding-модель от Microsoft Bing search-team. SOTA по retrieval-quality, semantic matching и contextual analysis.

**Ключевое:**
- #1 на multilingual MTEB-v2 leaderboard
- 100+ языков
- Длинный контекст до 32K input tokens
- Built for next-generation search workloads
