## Hive

Фреймворк для создания роёв AI-агентов с возможностью самосовершенствования. Агенты в рое могут улучшать свои собственные стратегии и инструменты в процессе работы.

**Возможности:**
- Оркестрация множества агентов (swarm)
- Самосовершенствование агентов на основе результатов
- Агентный tree search для решения сложных задач
