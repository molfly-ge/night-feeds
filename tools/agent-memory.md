# agent-memory (Neo4j Labs)

**Граф-нативная система памяти для AI-агентов** от Neo4j Labs.

Хранит и извлекает память агентов через граф знаний. В отличие от векторных баз данных, граф позволяет строить связи между сущностями и делать многошаговые рассуждения по памяти. Интегрируется с LangChain, LlamaIndex и другими AI-фреймворками.
