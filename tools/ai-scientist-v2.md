## AI-Scientist-v2

Проект SakanaAI — AI-система, которая самостоятельно генерирует научные статьи с помощью агентного tree search. Вторая версия с улучшенными возможностями генерации исследований.

**Возможности:**
- Автономная генерация workshop-papers
- Агентный tree search для планирования и написания
- Итеративное улучшение качества текста
