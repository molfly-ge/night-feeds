# Starcloud

**Тип:** Orbital Data Center / Space Computing
**Автор:** Philip Johnston
**Twitter:** [@Starcloud_io](https://x.com/StockSavvyShay/status/2038595321564176410)
**Финансирование:** $170M Series A, оценка $1.1B (YC, быстрейший единорог в истории)
**Источник:** [@StockSavvyShay](https://x.com/StockSavvyShay/status/2038595321564176410), [@aakashgupta](https://x.com/aakashgupta/status/2038647198607716476), 30-03-2026

Starcloud строит орбитальные дата-центры на низкой орбите Земли для решения энергетического кризиса AI-инфраструктуры.

**Достижения:**
- Starcloud-1: 60 кг, один NVIDIA H100, обучила AI-модель в космосе — первая в истории
- Старт за $3M, единорог за 17 месяцев с Demo Day
- Рекорд: крупнейший Series A за серию А в истории YC

**Starcloud-2 (в разработке):**
- 450 кг, 100× мощность генерации
- NVIDIA Blackwell B200, AWS server blade, bitcoin mining
- Клиенты: Crusoe, AWS, Google Cloud, NVIDIA

**Инвесторы:** Benchmark, EQT (70+ датацентров), Macquarie Capital ($500B AUM)

**Тезис:** LEO имеет неограниченную солнечную энергию и ноль NIMBY-оппозиции. Цель: $0.05/кВтч при стоимости запуска Starship $500/кг.
