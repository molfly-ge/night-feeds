## VisionClaw

Open-source проект, превращающий очки Meta Ray-Ban в голосового AI-агента. Потоковая передача видео (1fps) в реальном времени + голосовое взаимодействие через Gemini Live.

**Возможности:**
- Работает на iOS/Android (тестируется через камеру телефона)
- Полностью hands-free управление
- 56+ навыков OpenClaw для реальных задач: описание сцен, отправка сообщений, добавление в списки, поиск рядом
- Полностью open-source

**Технологии:**
- Gemini Live (мультимодальный AI)
- OpenClaw skills
