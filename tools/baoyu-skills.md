# baoyu-skills

Набор навыков для Claude Code, включая генерацию слайдов (baoyu-slide-deck) в различных стилях, в том числе рукописном.

## Ключевые возможности

- Генерация слайдов из контента
- Рукописный стиль и другие визуальные темы
- Интеграция с Claude Code
