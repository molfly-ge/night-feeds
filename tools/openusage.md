# OpenUsage

Минималистичный дашборд для аудита расходов на AI-подписки. Живёт в menu bar, показывает в процентах сколько потратил по подпискам кодинг-AI.

**Дизайн:** прогресс-бары, бейджи, понятные подписи. Без mental math — сразу видно, на сколько хватит подписки до конца месяца.
