# OpenHarness

Composable-слой управления для Claude Code и Codex. Делает AI-кодинг контролируемым — структурированные агенты поверх существующих инструментов.

## Ключевые возможности

- Composable агентная архитектура
- Совместимость с Claude Code и Codex
- Контроль и воспроизводимость AI-кодинга
