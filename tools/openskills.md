# OpenSkills

CLI-инструмент для портативности AI-навыков между агентами: Claude Code, Cursor и другие. Навыки, написанные один раз, работают в любом агенте.

## Ключевые возможности

- Переносимость навыков между AI-агентами
- CLI-интерфейс
- Поддержка Claude Code, Cursor и др.
