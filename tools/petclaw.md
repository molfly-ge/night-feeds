# PetClaw

1-click локальный AI-агент в формате персистентного «питомца» на десктопе. Отличается от других open-source агентских фреймворков тем, что обходит «configuration tax» — без debug gateway-ошибок, Python-зависимостей, API-ключей. По заявлению разработчиков, локальный AI-питомец готов исполнять задачи под минуту от установки.

«Питомец» живёт на экране, учит привычки пользователя и адаптируется под daily workflows.

**Демо:** автономный outreach pipeline — скрейп креаторов, отправка outbound emails, экспорт в structured sheet — всё в фоне без manual override.
