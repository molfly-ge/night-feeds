# marker-api

**Конвертер PDF в Markdown** с поддержкой распределённой обработки.

Превращает PDF-документы в чистый Markdown с сохранением структуры. Поддерживает распределённую обработку для больших объёмов. Полезен для подготовки данных для RAG-систем и документационных пайплайнов.
