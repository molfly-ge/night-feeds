# Deep Scientist

AI-студия для научных исследований, работающая на локальном оборудовании. Позволяет проводить эксперименты, анализировать данные и генерировать гипотезы с использованием LLM без отправки данных в облако. Ориентирована на исследователей, которым важна приватность и контроль над вычислениями.
