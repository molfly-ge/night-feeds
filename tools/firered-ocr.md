# FireRed OCR

**OCR-модель на 2B параметров** с лицензией Apache 2.0.

FireRed OCR — модель оптического распознавания текста с 2 миллиардами параметров. Открытая лицензия Apache 2.0, высокая точность распознавания на множестве языков и форматов документов.
