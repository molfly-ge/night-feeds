## lingo.dev

AI-инструментарий для автоматической локализации приложений. Мгновенный перевод и адаптация UI-текстов с помощью AI.

**Возможности:**
- Автоматическая локализация через AI
- Поддержка множества форматов (JSON, YAML, PO и др.)
- Интеграция в CI/CD pipeline
- CLI и API доступ
