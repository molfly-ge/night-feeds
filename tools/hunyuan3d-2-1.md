# Hunyuan3D-2.1 (Tencent)

**Опенсорс генерация 3D-ассетов с PBR-текстурами от Tencent.**

Модель Hunyuan3D версии 2.1 — создаёт полноценные 3D-объекты с физически корректными текстурами (PBR) из текстового описания или изображения. Для геймдева, VR/AR, промышленного дизайна.
