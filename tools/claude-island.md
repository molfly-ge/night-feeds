# Claude Island

macOS-приложение для управления сессиями Claude Code через Dynamic Island (область «чёлки» MacBook). Написано на Swift, работает через хуки в `~/.claude/hooks/` и Unix-сокеты.

## Возможности

- Одобрение/отклонение операций Claude Code прямо из Dynamic Island
- Просмотр полной истории диалогов с рендерингом Markdown
- Одновременное управление несколькими сессиями Claude Code
- Автоматическая настройка хуков при первом запуске

## Требования

- macOS 15.6+
- Claude Code
