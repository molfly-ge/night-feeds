# Midday CLI

CLI-бэкбон для бизнес-операций через AI-агентов.

**Возможности:** 80+ инструментов — инвойсинг, сверка, экспорты, тайм-трекинг, отчёты. Позволяет агентам управлять финансами компании через единый интерфейс.
