# memory-wiki (OpenClaw plugin)

Плагин для OpenClaw, который превращает память агента в реальный knowledge layer — а не «vibes». Релиз в составе OpenClaw 2026.4.7.

**Что внутри:**
- Structured claims с evidence
- Tracking противоречий и свежести (contradiction + freshness tracking)
- Compiled digests для агентов
- Bridge mode из активных memory-плагинов
- Опционально Obsidian-friendly vaults

Тег от команды OpenClaw: «потому что “trust me bro” не является системой знаний».
