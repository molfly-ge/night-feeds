## claude-code-video-toolkit

Skill (навык) для Claude Code, позволяющий генерировать и редактировать видеоконтент прямо из среды разработки.

**Возможности:**
- Создание видеоматериалов через Claude Code без выхода из IDE
- Интеграция с рабочим процессом разработчика
- Основан на инфраструктуре digitalsamba
