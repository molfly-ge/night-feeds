# MOSS-TTS

Мультиязычная text-to-speech модель от OpenMOSS. Поддерживает 6 языков. 71k+ загрузок на Hugging Face. Подходит для создания голосовых приложений с мультиязычными требованиями.

**Ссылки:**
- Hugging Face: https://huggingface.co/OpenMOSS-Team/MOSS-TTS
