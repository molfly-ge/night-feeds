# Voxtral-Mini-4B-Realtime

Модель распознавания речи в реальном времени от Mistral AI. Компактный 4B-параметрный пакет с мультиязычной поддержкой ASR (automatic speech recognition). Около 900k загрузок.

**Тип:** ASR (Automatic Speech Recognition)
**Разработчик:** Mistral AI
**Параметры:** 4B
**Загрузки:** ~900k

## Ключевые особенности

- Real-time мультиязычная транскрипция речи
- Компактный размер (4B) при высоком качестве
- Подходит для edge-деплоя
