# Cline Kanban

**Тип:** Multi-Agent Orchestration
**Источник:** @roundtablespace, 27-03-2026

Standalone-приложение от команды Cline для оркестрации мульти-агентных задач.

**Особенности:**
- Задачи выполняются в изолированных worktrees
- Просмотр и ревью кода в один клик
- Визуальная Kanban-доска для управления агентами

**Для кого:** разработчики, запускающие несколько AI-агентов параллельно и нуждающиеся в визуальном контроле.
