# Chronos Forecasting

**Тип:** Библиотека / претренированные модели для прогнозирования временных рядов  
**GitHub:** https://github.com/amazon-science/chronos-forecasting  
**Организация:** Amazon Science

## Описание

Chronos — набор претренированных моделей для zero-shot и few-shot прогнозирования временных рядов, основанных на архитектурах языковых моделей (transformer). Разработан Amazon Science.

## Ключевые возможности

- Претренированные модели для прогнозирования временных рядов без дополнительного обучения
- Архитектура на основе language model transformers
- Zero-shot прогнозирование на новых данных
- Различные размеры моделей (tiny, mini, small, base, large)

## Применение

- Прогнозирование продаж, трафика, потребления ресурсов
- Финансовые временные ряды
- IoT-данные и метрики мониторинга
