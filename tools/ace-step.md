# Ace-Step

Модель генерации музыки из текста (text-to-audio) от команды ACE-Step. Около 40k загрузок, баланс между качеством и доступностью.

**Тип:** Text-to-audio / music generation
**Разработчик:** ACE-Step
**Версия:** Ace-Step1.5
**Загрузки:** ~40k

## Ключевые особенности

- Генерация музыки по текстовому описанию
- Открытая модель
- Высокое качество аудио при разумном размере
