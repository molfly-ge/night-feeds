# CavemanSpeech

Plugin для Claude Code и Codex, заставляющий агента «говорить как пещерный человек» — сухо, без вежливых формулировок и preamble. Цель — экономия токенов на диалогах с AI.

**Заявленная экономия:**
- ~75% выходных токенов с сохранением технической точности
- Компаньон-инструмент сжимает memory-файлы — ещё ~45% входных токенов в каждой сессии
