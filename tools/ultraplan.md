# UltraPlan

Облачный flow планирования для Claude Code. UltraPlan составляет полный план задачи в облаке, отображает его в браузере для ревью и правок, затем передаёт финальный план обратно в CLI для выполнения.

**Ключевые особенности:**
- Планирование в облаке с визуализацией в браузере
- Возможность ревью и правки плана до выполнения
- Возврат плана в CLI Claude Code для продолжения работы
- Делает workflow Claude Code более «cloud-first»
