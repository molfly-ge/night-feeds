# Oh My Codex (OmX)

Расширение для OpenAI Codex CLI, добавляющее хуки, командные агенты (agent teams), HUD-панели и другие улучшения рабочего процесса. Проект набрал около 14.7K звёзд на GitHub и активно развивается как надстройка над стандартным Codex CLI.
