# marknative

**Рендеринг Markdown в PNG/SVG без браузера, без Chromium, без DOM.**

Раньше: Markdown → HTML → headless Chrome → скриншот. Теперь: один шаг. Полезно для генерации карточек, превью, документации и любых pipeline, где нужно изображение из текста, но запускать браузер слишком тяжело.
