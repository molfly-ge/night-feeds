# Career-Ops

Open-source система автоматизации поиска работы. Playwright сканирует вакансии, фильтрует по релевантности, отслеживает outreach и follow-up в пайплайне.

## Ключевые возможности

- 14 режимов навыков на Claude Code
- Go-дашборд для мониторинга
- Генерация PDF, пакетная обработка
- Полный цикл: скан → фильтрация → заявка → follow-up
