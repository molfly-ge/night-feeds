# XLeRobot

Двурукий мобильный робот за **$660** с 4-часовой сборкой. Open hardware — спецификации, прошивки, BOM открыты.

Цель проекта — сделать робототехнику доступной для тех, у кого нет $30k+ на индустриальный манипулятор. Подходит для исследований, прототипирования, обучения и хобби.
