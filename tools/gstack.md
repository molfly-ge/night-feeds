# GStack

**Тип:** Claude Code Skills / Configuration Layer
**Автор:** Garry Tan (@garrytan), CEO Y Combinator
**Twitter:** [@garrytan](https://x.com/garrytan)
**Источник:** [@garrytan](https://x.com/garrytan), 29-03-2026

GStack — набор навыков, правил и конфигураций для Claude Code, создающий «второй слой» поверх стандартного Claude Code. Устанавливается прямо из Claude Code: «скажи окну Claude Code установить GStack».

**Навыки:**
- `/design-consultation` — голосовое взаимодействие с Claude Code для дизайн-решений
- `/codex` — кодовые ревью, в т.ч. adversarial
- `/tasks` — экран мониторинга прогресса агентных заданий

**Принципы (ETHOS.md):**
- **User Sovereignty**: AI-модели рекомендуют. Пользователи решают. Это правило имеет приоритет над всеми остальными.

**Особенности:**
- Постоянно развивается: автор публикует ежедневные обновления
- Ограничение автономии: модель не переключает задачи без согласования с пользователем
- Совместим с YC CEO / Design / Engineering контекстами
