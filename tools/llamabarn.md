# LlamaBarn

GUI-приложение для управления моделями llama.cpp от организации ggml-org. Упрощает загрузку, организацию и запуск GGUF-моделей через графический интерфейс вместо командной строки. Полезен для пользователей, работающих с локальными LLM.
