# Bullpen CLI

**Тип:** CLI-инструмент / платформа для предиктивных рынков  
**Сайт:** https://cli.bullpen.fi/  
**Платформа:** https://go.bullpen.fi/  
**LLM-документация:** https://cli.bullpen.fi/llm.txt

## Описание

Bullpen — платформа для работы с предиктивными рынками (prediction markets), прежде всего Polymarket. Bullpen CLI — командная строка для автоматизации торговли, получения данных о топ-трейдерах и построения AI-ботов.

## Ключевые возможности

- Получение данных о лучших трейдерах предиктивных рынков
- Задание правил и стратегий для автоматической торговли
- Создание дашбордов для мониторинга
- Совместимость с Claude Code и LLM-агентами (llm.txt)
- Интеграция с Polymarket

## Применение

Пример: Sharbel (@sharbel) собрал Polymarket-бота за одну сессию Claude Code с использованием Bullpen CLI, достигнув 13% прибыли за 3 часа.
