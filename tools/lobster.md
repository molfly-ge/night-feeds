# Lobster (@clawdbot/lobster)

Встраиваемое ядро runtime для OpenClaw — позволяет запускать агентов не только как standalone-приложение, но и встраивать движок в другие проекты. Версия 2026.4.6 добавила структурированные паузы ввода, расширенные условия рабочих процессов, безопасные dry-run и исправления для Windows.

## Ключевые возможности

- Embeddable core runtime — встраивается в любое Node.js приложение
- Structured input pauses — агент может запрашивать ввод пользователя в структурированном формате
- Richer workflow conditions — расширенная логика условий в TaskFlow
- Safer dry-runs — безопасный режим проверки без выполнения действий
- Windows-compatible — улучшенная совместимость с Windows
