# GBrain

Персональная RAG-система для OpenClaw на базе PostgreSQL pgvector. Семантический поиск по десяткам тысяч markdown-файлов — личная база знаний для AI-агента.

## Ключевые возможности

- Полная pgvector-имплементация для векторного поиска
- Интеграция с OpenClaw — установка одной командой
- Вдохновлён LLM Wiki Карпати, расширен собственными навыками
- MIT-лицензия
