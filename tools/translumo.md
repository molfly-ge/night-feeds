# Translumo

**Опенсорс инструмент для перевода текста на экране в реальном времени.**

Читает текст с любой области экрана через OCR и переводит его прямо поверх изображения. Работает офлайн, поддерживает несколько движков перевода. Полезно для игр, субтитров, документации на иностранных языках.
