# Gemini CLI

**Терминальный AI-агент от Google** для работы с Gemini прямо из командной строки.

Официальный CLI от Google DeepMind. Позволяет использовать Gemini в терминале: читать файлы, выполнять задачи, общаться с моделью. Часть экосистемы Google Gemini Developer Tools.
