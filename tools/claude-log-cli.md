# claude-log-cli

CLI для поиска по логам Claude Code через native AOT binary. Запускается мгновенно — без рантайма JIT, без интерпретатора.

Удобно, когда нужно быстро отыскать конкретную команду, инструмент или ошибку из истории Claude Code сессий.
