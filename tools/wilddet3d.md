# WildDet3D

Open-модель для monocular 3D object detection «в дикой природе» — то есть на произвольных RGB-изображениях без лидара и стерео.

**Что умеет:**
- Принимает на вход текстовые промпты, клики или 2D-боксы
- На zero-shot evaluations почти удваивает прежние best scores
