# lark-cli

Open source CLI от Feishu (飞书) для управления платформой через AI-агентов. Позволяет агенту отправлять сообщения, просматривать календарь, писать документы, создавать таблицы и отправлять письма.

Часть тренда 2026: все enterprise-платформы делают CLI для поддержки AI-агентов (Google gws, Microsoft и др.).
