## OpenWolf

Middleware-прослойка для Claude Code, реализующая концепцию «второго мозга» — долгосрочной памяти и контекста для AI-ассистента.

**Возможности:**
- Persistent memory для Claude Code сессий
- Сохранение и извлечение контекста между сессиями
- Middleware-архитектура — встраивается в существующий рабочий процесс
