# TRL (Transformer Reinforcement Learning)

**Библиотека HuggingFace для обучения LLM с подкреплением.** Версия 1.0 вышла 01-04-2026.

TRL реализует RLHF, PPO, DPO, ORPO и другие методы alignment. v1.0 принёс стабильные API, широкие интеграции с экосистемой HuggingFace и архитектуру, рассчитанную на будущие изменения в области.
