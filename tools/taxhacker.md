# TaxHacker

**Самохостируемый AI для обработки чеков и счетов** с автоматической категоризацией.

TaxHacker — self-hosted инструмент на базе AI для обработки чеков, инвойсов и финансовых документов. Автоматически извлекает данные, категоризирует расходы. Полностью локальный, без передачи данных третьим сторонам.
