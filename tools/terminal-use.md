# terminal-use

**Headless виртуальный терминал для AI-агентов.**

Позволяет AI-агентам взаимодействовать с CLI-инструментами без реального TTY. Агент получает полноценное терминальное окружение в памяти — запускает команды, читает вывод, реагирует на интерактивные приглашения.
