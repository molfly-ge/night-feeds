# ccremote

**Удалённые апрувы Claude Code через Discord.**

Инструмент, позволяющий получать уведомления от Claude Code в Discord и апрувить или отклонять запросы прямо из мессенджера. Решает проблему необходимости сидеть рядом с терминалом во время долгих agentic-задач.
