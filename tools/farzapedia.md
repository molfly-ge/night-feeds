# Farzapedia

Персональная вики, созданная LLM на основе личных данных: дневниковых записей, заметок, переписки. Из 2500 записей LLM сформировала 400 связанных статей о людях, проектах, темах интересов с обратными ссылками.

Подход придерживается четырёх принципов: явность (память прозрачна и инспектируема), владение данными (хранится локально), универсальный формат (markdown + изображения), независимость от провайдера ИИ (BYOAI).
