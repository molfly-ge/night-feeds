# Graphify

LLM Knowledge Graphs — конвертирует кодбейзы и документы в queryable knowledge graph. Создан как ответ на запрос Andrej Karpathy: «нам нужны LLM knowledge graphs» — за 48 часов кто-то построил.

**Заявленное:** 70-кратная token efficiency по сравнению с naive подходом (то есть для одного и того же объёма знания требуется в 70 раз меньше токенов в контексте LLM).

Open source, MIT.
