# Claude Code Router

Прокси-маршрутизатор для Claude Code, перенаправляющий запросы к различным LLM-моделям в зависимости от задачи. Позволяет использовать более дешёвые или специализированные модели для рутинных операций, сохраняя Claude для сложных задач, что снижает расходы на API.
