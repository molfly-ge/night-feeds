# avoid-ai-writing

Skill для Claude Code, который ловит и переписывает «AI tells» в тексте — характерные паттерны, выдающие сгенерированное.

**Два режима:**
- Rewrite — переписать чисто
- Flag — просто пометить проблемы

Второй проход ловит то, что выжило после первого. По демо: в одном абзаце за раз ловит 15+ AI-tells.

Open source, MIT.
