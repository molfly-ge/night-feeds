# OpenScreen

Бесплатный open source инструмент для создания демонстрационных видео и скринкастов — альтернатива Screen Studio. Без подписок, без водяных знаков, разрешено коммерческое использование.

## Ключевые возможности

- Создание красивых демо-видео и скринкастов
- Без ограничений и водяных знаков
- Разрешено коммерческое использование
- Open source, бесплатно
