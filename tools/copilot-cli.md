# Copilot CLI

CLI-инструмент от GitHub для работы с GitHub Copilot прямо из терминала. Позволяет генерировать команды, объяснять код и взаимодействовать с Copilot без необходимости открывать IDE. Ориентирован на разработчиков, предпочитающих терминальный workflow.
