# Shannon

Инструмент автоматического пентеста веб-приложений. Запускается одной командой (`npx @keygraph/shannon`), клонирует репозиторий и автоматически проверяет: SQL-инъекции в формах входа, XSS в полях ввода, SSRF через внутренние запросы, корректность реализации JWT, обходы аутентификации в admin-панелях.

**Ключевые факты:**
- 36k+ звёзд на GitHub
- Не требует знаний в области безопасности
- Генерирует отчёт автоматически
