# Exo Email

**Open-source email-клиент на базе Claude Code** с CLI-инструментами.

Exo Email — открытый email-клиент, построенный на Claude Code. Включает CLI-инструменты для работы с почтой из терминала. Версия 0.6.0 с улучшенной интеграцией AI-ассистента.
