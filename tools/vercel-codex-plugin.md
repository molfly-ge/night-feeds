# Vercel Plugin for Codex

Плагин Vercel в приложении OpenAI Codex. Позволяет пройти полный путь от настройки проекта до деплоя прямо внутри интерфейса Codex. Упрощает рабочий процесс разработчиков, интегрируя деплой-платформу Vercel в AI-среду Codex.
