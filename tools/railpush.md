# RailPush

**PaaS-платформа для деплоя AI-агентов** с 235 MCP-инструментами и GPU-вычислениями.

RailPush — платформа как сервис для AI-агентов. Предоставляет 235 готовых MCP-инструментов, управляемый Postgres, GPU-вычисления и деплой через git push. Упрощает запуск и масштабирование агентных приложений.
