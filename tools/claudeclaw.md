# claudeclaw

**Постоянный оркестратор агентов для Claude Code.**

Управляет несколькими агентами Claude Code как постоянными воркерами: сохраняет состояние между сессиями, распределяет задачи, следит за прогрессом. Решает проблему эфемерности стандартных Claude Code-сессий.
