# Atomic Chat

**Опенсорс альтернатива ChatGPT** с поддержкой локальных LLM и облачных моделей.

Позволяет работать с локальными моделями через Ollama, подключать облачные API (OpenAI, Anthropic), сохраняя полный контроль над данными. Поддерживает смешанный режим: локальная обработка + облачные модели по необходимости.
