# Browser Use CLI 2.0

**Тип:** CLI / Browser Automation
**GitHub:** https://github.com/browser-use/browser-use
**Twitter:** https://x.com/browser_use
**Источник:** [@roundtablespace](https://x.com/roundtablespace), 27-03-2026

Инструмент для прямого управления браузером из командной строки. Версия 2.0 — 2x скорость, половина стоимости.

**Возможности:**
- Прямое управление браузером (не через Selenium/Playwright обёртки)
- Облачный режим и self-hosted
- Cloud Platform Management
- Интеграция с AI-агентами
