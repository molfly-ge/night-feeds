# Clicky

AI-репетитор, работающий прямо рядом с курсором. Clicky видит экран пользователя, разговаривает голосом и указывает на конкретные элементы — как живой преподаватель, сидящий рядом.

**Ключевые особенности:**
- Видит экран пользователя в реальном времени
- Голосовое взаимодействие
- Указывает на конкретные элементы интерфейса (как реальный тьютор)
- Работает вместе с любым приложением
