# Sherlock

OSINT-инструмент для поиска профилей пользователя по имени на 400+ платформах. Одной командой `sherlock <username>` находит аккаунты на Instagram, X, TikTok, Reddit, GitHub, LinkedIn и сотнях других сайтов. Применяется для пентестинга, red team операций и сбора разведывательных данных.

Поддерживает: Tor-анонимность, HTTP-прокси, экспорт результатов в CSV/Excel, автоматическое открытие найденных профилей в браузере. 78k+ звёзд на GitHub.
