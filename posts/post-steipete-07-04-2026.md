16 твитов за сутки. Главные темы — переход OpenClaw-сообщества с Claude на GPT-5.4/Codex, GitHub становится официальным спонсором OpenClaw, выход Lobster 2026.4.6, а также OpenAI Codex для open source.

**GitHub — официальный спонсор OpenClaw** — GitHub стал официальным спонсором OpenClaw, предоставив команде Copilot Pro+, финансирование безопасности и ресурсы для масштабирования. OpenClaw называют самым быстрорастущим open source проектом в истории. [Твит](https://x.com/steipete/status/2041254442994205164)

**Переход с Claude на Codex/GPT-5.4** — многие пользователи OpenClaw переходят с Claude OAuth на Codex OAuth. Один пользователь сообщает, что запускает команду из 12 агентов через 1 аккаунт GPT-5.4/Codex за $200/месяц без рейт-лимитов, тогда как раньше требовалось 4 аккаунта Claude Pro Max. [Твит](https://x.com/steipete/status/2041272727613665517)

**Lobster 2026.4.6** — вышел новый релиз ядра OpenClaw: встраиваемый runtime, структурированные паузы ввода, расширенные условия рабочих процессов, более безопасные dry-run и исправления для Windows. [Твит](https://x.com/steipete/status/2041168951950925839) · [npm: @clawdbot/lobster](https://www.npmjs.com/package/@clawdbot/lobster)

**OpenAI Codex for Open Source** — OpenAI опубликовал обновление программы Codex для open source: более 100 крупных проектов получают поддержку, включая Linux, React, Node.js, Rust, Python, Kubernetes, Flutter, Electron, Ollama и многие другие. **Claude Code — самозащита от анализа** — Claude Code теперь выдаёт ошибку, если попытаться использовать его для анализа собственного исходного кода. [Твит](https://x.com/steipete/status/2041016477047034012)
