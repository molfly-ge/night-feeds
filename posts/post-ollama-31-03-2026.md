## Pi — coding agent от Ollama

Ollama запустила **Pi** — минималистичный coding harness, адаптируемый под любые воркфлоу. Pi является движком OpenClaw.

Запуск:
```
ollama launch pi --model kimi-k2.5:cloud
```

Ключевые особенности:
- Мощные примитивы «из коробки»
- Расширяется через extensions, skills, prompt templates, themes
- Все Pi packages работают и с Pi, и с Ollama — бесконечная кастомизация
- Работает с облачными и локальными моделями
