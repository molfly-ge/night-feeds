# brandgrowthos — 28-03-2026

Практическая находка по браузерной автоматизации: для форм с динамически меняющимися ID полей помогло не улучшение селекторов, а пересканирование DOM после каждого действия пользователя.

Главная тема — Claude Mythos и утечка из Anthropic. Внутренний документ команды безопасности описывает следующую модель как несущую «беспрецедентные риски кибербезопасности». Утечка затронула ~3000 активов (PDF, изображения, детали мероприятий CEO), которые были открыты без аутентификации. Вывод для команд, строящих на Claude: готовьтесь к значительному обновлению возможностей и не рассматривайте провайдера LLM как чёрный ящик.

Тезис об автоматизации: категория «workflow automation» устаревает — агентам нужны цели и инфраструктура, а не воркфлоу. Большинство платформ автоматизации решают не ту проблему.

LlamaAgents Builder от LlamaIndex снижает барьер входа в агентов для обработки документов — от промпта до задеплоенного агента за минуты.
