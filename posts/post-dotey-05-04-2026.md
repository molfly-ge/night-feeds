# dotey — 05-04-2026

5 твитов за сутки.

**Anthropic отключает Claude-подписки от сторонних инструментов** — Claude Code 负责人 Boris Cherny объявил: с 12:00 PT 4 апреля Claude Pro и Max подписки больше не покрывают использование через сторонние инструменты типа OpenClaw. Альтернативы: дополнительные пакеты трафика (со скидкой) или API-ключ с оплатой по факту. Существующим подписчикам начислен разовый кредит, равный месячной плате; по запросу — полный возврат. Anthropic последовательно закрывал эту возможность с января 2026: сначала технически заблокировал имитацию Claude Code сторонними инструментами, затем в феврале обновил ToS. [Твит](https://x.com/dotey/status/2040239433362862323)

**claude -p тоже считается сторонним использованием** — Оказалось, что флаг `-p` (pipe/programmatic mode) у Claude Code с 4 апреля тоже классифицируется Anthropic как «использование через третью сторону» и требует Extra Usage. Это стало неожиданностью для разработчиков, использующих его в скриптах. [Твит](https://x.com/dotey/status/2040281402906738746)

**Уточнение по -p** — Позднее dotey подтвердил, что флаг `-p` в итоге должен работать нормально. [Твит](https://x.com/dotey/status/2040505908615090277)

**Обход водяных знаков через Gemini** — dotey описывает промпт-технику: вместо прямого запроса на удаление водяного знака — просить Gemini заменить фон/текст другим цветом, затем вернуть оригинальные элементы. Подход позволяет обойти ограничения модерации. [Твит](https://x.com/dotey/status/2040260984703520792)

**DeepSeek V4 будет работать на чипах Huawei** — По данным The Information, DeepSeek специально отложил выход V4, чтобы портировать его на чипы Huawei Ascend 950PR (Atlas 350). Характеристики чипа: производительность в 2.87× выше H20, 112 ГБ HBM, пропускная способность 1.4 ТБ/с. Ключевая особенность — единственный китайский чип с поддержкой FP4 инференса, что снижает потребности по памяти в 4 раза. DeepSeek не предоставил ранний доступ американским чипмейкерам — впервые отдав приоритет Huawei и Cambricon. [Твит](https://x.com/dotey/status/2040236453528064241)
