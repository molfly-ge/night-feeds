# huggingmodels — 05-04-2026

50 твитов за сутки. Аккаунт публикует обзоры моделей с Hugging Face тройками: анонс → ценность → ссылка.

**3D-Speaker-MT.Axera** — мультиязычная аудио-в-текст модель, обрабатывающая English и Chinese из единой модели. Упрощает пайплайны для глобальных голосовых приложений. [Твит](https://x.com/HuggingModels/status/2040626239627763791) · [Модель](https://huggingface.co/AXERA-TECH/3D-Speaker-MT.Axera)

**Gemma3-CFG** — файн-тюн Google Gemma-3-12B, оптимизированный для управляемой генерации (controlled generation) с улучшенным следованием инструкциям. [Твит](https://x.com/HuggingModels/status/2040618687833813211) · [Модель](https://huggingface.co/amberyzheng/gemma3-cfg_seed_0_500)

**Llama-Guard-4-12B (RedHatAI)** — модель безопасности для фильтрации вредоносного контента в AI-диалогах. 8-битная квантизация снижает требования к железу при сохранении точности. [Твит](https://x.com/HuggingModels/status/2040611139349069990) · [Модель](https://huggingface.co/RedHatAI/Llama-Guard-4-12B-quantized.w8a8)

**MiniMax-M2.5** — текстовая модель для диалога, 637k+ загрузок. Поддерживает сложные диалоговые сценарии, совместима с популярными inference-эндпоинтами. [Твит](https://x.com/HuggingModels/status/2040596039271600317) · [Модель](https://huggingface.co/MiniMaxAI/MiniMax-M2.5)

**Qwen3.5-397B-A17B** — мультимодальная vision-language модель на 397B параметров, 1M+ загрузок. Понимает изображения и текст. [Твит](https://x.com/HuggingModels/status/2040588490027766053) · [Модель](https://huggingface.co/Qwen/Qwen3.5-397B-A17B)

**MOSS-TTS** — мультиязычный text-to-speech от OpenMOSS, 71k+ загрузок, поддержка 6 языков. [Твит](https://x.com/HuggingModels/status/2040580940272173099) · [Модель](https://huggingface.co/OpenMOSS-Team/MOSS-TTS)

**Llama 3.3 70B Instruct** — мощная открытая модель Meta для диалога и генерации кода, 400k загрузок. [Твит](https://x.com/HuggingModels/status/2040543191074783396) · [Модель](https://huggingface.co/meta-llama/Llama-3.3-70B-Instruct)

**Meta-Llama-3.1-8B-Instruct-GGUF (bartowski)** — квантизованная версия GGUF для локального запуска, 240k загрузок. Поддерживает English, German, French. [Твит](https://x.com/HuggingModels/status/2040535641642213557) · [Модель](https://huggingface.co/bartowski/Meta-Llama-3.1-8B-Instruct-GGUF)

**Dolphin3.0-Llama3.1-8B** — специализированный файн-тюн на рассуждение, математику и вызов функций (function calling) с Chain-of-Thought обучением. [Твит](https://x.com/HuggingModels/status/2040528091295203716) · [Модель](https://huggingface.co/dphn/Dolphin3.0-Llama3.1-8B)

**Llama 3.2/3.1 серия** — также обозревались модели Llama 3.2-3B-Instruct (6.3M загрузок), Llama 3.2-1B, Llama 3.2-1B-Instruct (4M загрузок), Llama 3.1-8B и Llama 3.1-70B. [Модели Meta Llama](https://huggingface.co/meta-llama)
