# ollama — 05-04-2026

17 твитов за сутки.

**Ollama открыла облако для пользователей OpenClaw** — После того как Anthropic запретила использование Claude-подписок в OpenClaw, Ollama объявила о сбросе лимитов использования для своих подписчиков. Цель — обеспечить плавный переход для пользователей сторонних инструментов. [Твит](https://x.com/ollama/status/2040321358853537920)

**Призыв к миграции** — Ollama предложила OpenClaw-пользователям перейти на свой облачный сервис ($20/мес.). Доступные модели: kimi-k2.5:cloud, glm-5:cloud, minimax-m2.7:cloud — запускаются командой `ollama launch openclaw`. [Твит](https://x.com/ollama/status/2040261847597867159)

**Скорость GLM-5 в облаке Ollama** — Пользователи сообщают о 91,9 токена в секунду на модели GLM-5 в облаке Ollama. [Твит](https://x.com/ollama/status/2040477538137583689)

**OpenClaw переходит в OpenClaw Foundation** — Peter Steinberger (@steipete) подтвердил, что OpenClaw будет передан в некоммерческий OpenClaw Foundation — вне контроля любой компании. Ollama сделала ретвит. [Твит](https://x.com/ollama/status/2040501094699487413)

**Регистрация аккаунта Ollama** — Анонс страницы регистрации облачного сервиса. [Твит](https://x.com/ollama/status/2040261849586041213) | [Сайт](https://ollama.com/)
