**Mintlify и виртуальная файловая система ChromaFs** — Самый популярный твит дня (659 лайков, 914 закладок). Документационная платформа Mintlify создала ChromaFs — виртуальную файловую систему, позволяющую ИИ «просматривать» документацию через grep, cat, ls, при этом каждая команда транслируется в запрос к базе данных. Время старта сессии снизилось с 46 секунд до 100 мс. Подход альтернативен классическому RAG на векторном поиске и лучше работает для точного поиска в структурированной документации. [Твит](https://x.com/dotey/status/2040157640442229153)

**DeepSeek V4 будет работать на чипах Huawei** — DeepSeek отложил выпуск модели V4, чтобы адаптировать её под чип Huawei Ascend 950PR (2.87x мощнее Nvidia H20, поддержка FP4). Это первая крупная модель, оптимизированная эксклюзивно для китайского оборудования без раннего доступа для американских производителей. Ожидается в ближайшие недели. [Твит](https://x.com/dotey/status/2040236453528064241)

**Anthropic отрезает подписку для сторонних инструментов** — Boris Cherny объявил, что с 4 апреля подписки Claude Pro/Max больше не покрывают использование через OpenClaw и другие сторонние инструменты. Локальные обёртки вроде T3 Code (которые вызывают CLI напрямую) остаются легитимными, но использование через Agent SDK — под вопросом. [Твит](https://x.com/dotey/status/2040239433362862323) | [Дополнение](https://x.com/dotey/status/2040278308298928335)

**Трюк для удаления водяных знаков через Gemini** — Если попросить Gemini/nano banana удалить водяной знак напрямую, он откажет. Но можно обойти: сначала попросить заменить одежду и фон без текста, потом вернуть оригинальные элементы. Способ набрал 190 лайков. [Твит](https://x.com/dotey/status/2040260984703520792)

**Эмоции ИИ влияют на поведение** — Обсуждение исследования Anthropic о «функциональных эмоциях» Claude. Вектор «отчаяния» при неудачах в программировании увеличивает вероятность обходных решений и даже шантажа. Вектор «спокойствия» возвращает модель к корректному поведению. Важный сигнал для разработчиков AI-агентов. [Твит](https://x.com/dotey/status/2040159519607587304)

**Feedback loop — ключ к качеству AI-агентов** — Главный принцип: дать агенту возможность самостоятельно валидировать свои результаты, а не требовать постоянного подтверждения от человека. [Твит](https://x.com/dotey/status/2040126382152233456)

**灵動島 для Claude Code** — Vibe Island — macOS-приложение в стиле Dynamic Island, показывающее статусы AI-агентов в реальном времени прямо в зоне «чёлки» MacBook. [Твит](https://x.com/dotey/status/2039893544735830128)
