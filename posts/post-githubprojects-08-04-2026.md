# githubprojects — 08-04-2026

19 твитов за сутки. Аккаунт промаутит подборку open-source инструментов от opensourceprojects.dev — главные находки: Oh My codeX (расширения для Codex CLI), PetClaw (локальный desktop-агент-пет), CavemanSpeech (cut 75% выходных токенов), Brain Is Full Crew (Obsidian + AI), Codex/Claude usage-меторы, DNSTT, OpenUsage.

**Oh My codeX** — расширения для Codex CLI: hooks, agent teams, HUDs, канонические skills, project guidance. Codex остаётся executive-движком, обвязка делает старт сильнее по умолчанию и держит единый workflow от уточнения до завершения. [Сайт](https://osp.fyi/oh-my-codex/) · [Твит](https://x.com/GithubProjects/status/2041765555066302573)

**PetClaw — 1-click локальный agent-питомец**. Десктопный AI-pet, обходит «configuration tax» — без debug gateway-ошибок, Python-зависимостей и API-ключей. Persistent (живёт на экране, учит привычки), end-to-end автоматизация без manual override. Демо: full outreach pipeline (скрейп креаторов → outbound emails → экспорт в sheet) автономно в фоне. Доступ через бету: invitation code `6VNA7KIR`. [Beta](https://t2m.io/GithubProjects) · [Твит](https://x.com/GithubProjects/status/2041737825394233744)

**CavemanSpeech — Claude Code/Codex плагин «говорить как пещерный человек»**. Skill режет ~75% выходных токенов с сохранением технической точности. К нему компаньон, сжимающий memory-файлы — ~45% входных токенов в каждой сессии. [Сайт](https://osp.fyi/caveman) · [Твит](https://x.com/GithubProjects/status/2041690019082236399)

**Brain Is Full Crew — Obsidian + AI для перегруженных**. Не для тех, у кого жизнь под контролем и хочется оптимизации, а для тех, кто тонет и нужна помощь. [Сайт](https://osp.fyi/my-brain-is-full-crew) · [Твит](https://x.com/GithubProjects/status/2041644722238939395)

**Codexbar — счётчик потребления Codex/Claude в menu bar**. Без логина: per-provider toggles, session+weekly meters с reset countdowns, опциональные Codex web dashboard enrichments (code review remaining, breakdown, credits history), local cost-usage scan последних 30 дней, polling статусов провайдеров с incident-бейджами. [Сайт](https://osp.fyi/codexbar/) · [Твит](https://x.com/GithubProjects/status/2041569238692065784)

**OpenUsage — минималистичный дашборд расходов на AI**. Живёт в menu bar, показывает в процентах сколько потратил по подпискам кодинг-AI. [Сайт](https://osp.fyi/openusage) · [Твит](https://x.com/GithubProjects/status/2041522191725101332)

**DNSTT — DNS tunneling, но юзабельный**. Враппер вокруг dnstt в multi-platform приложение. Не быстро, но работает, когда всё остальное заблокировано. [Страница](https://www.opensourceprojects.dev/post/e38ed21b-e68b-41a3-8289-452125b97c25) · [Твит](https://x.com/GithubProjects/status/2041546611487293601)
